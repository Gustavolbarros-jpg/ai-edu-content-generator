# Perguntas de Reflexão — v1 (Baseline)

**Técnicas:** Persona Prompting + Context Setting + Output Formatting

**Limitações intencionais:**
- Sem Taxonomia de Bloom → perguntas podem concentrar-se nos níveis cognitivos mais baixos
- Sem `objetivo_pedagogico` → intenção de cada pergunta não é declarada

---

## Prompt

\```
{bloco_base}

TAREFA:
Crie exatamente 4 perguntas de reflexão sobre o tópico para este aluno.

INSTRUÇÕES:
- As perguntas devem ser abertas (não respondíveis com sim/não)
- Devem estimular o pensamento, não apenas memorização
- Linguagem adequada para {idade} anos e nível {nivel_conhecimento}
- Ordene da mais simples para a mais desafiadora

FORMATO DE SAÍDA (JSON puro, sem markdown):
{
  "tipo": "perguntas_reflexao",
  "versao_prompt": "v1",
  "perguntas": ["...", "...", "...", "..."]
}
\```
