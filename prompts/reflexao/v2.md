# Perguntas de Reflexão — v2 (Otimizado)

**Técnicas:** Persona + Context Setting + **Taxonomia de Bloom** + Output Formatting

**Ganho vs v1:** `nivel_cognitivo` e `objetivo_pedagogico` explícitos garantem distribuição pelos 4 níveis de Bloom. Sem essa instrução o modelo concentra tudo nos níveis 1-2.

---

## Prompt

\```
{bloco_base}

TAREFA:
Crie exatamente 4 perguntas de reflexão sobre o tópico, com dificuldade crescente.

ESTRUTURA OBRIGATÓRIA:
  Pergunta 1 (compreensão): verifica se o aluno entendeu o conceito básico
  Pergunta 2 (aplicação): pede para o aluno usar o conceito em uma situação
  Pergunta 3 (análise): pede para o aluno comparar, diferenciar ou explicar por quê
  Pergunta 4 (síntese): pede para o aluno criar, imaginar ou propor algo novo

REGRAS:
- Perguntas abertas — não podem ser respondidas com "sim" ou "não"
- Linguagem adequada para {idade} anos e nível {nivel_conhecimento}

FORMATO DE SAÍDA OBRIGATÓRIO (JSON puro, sem markdown):
{
  "tipo": "perguntas_reflexao",
  "versao_prompt": "v2",
  "perguntas": [
    {"numero": 1, "nivel_cognitivo": "compreensão", "pergunta": "...", "objetivo_pedagogico": "..."},
    {"numero": 2, "nivel_cognitivo": "aplicação", "pergunta": "...", "objetivo_pedagogico": "..."},
    {"numero": 3, "nivel_cognitivo": "análise", "pergunta": "...", "objetivo_pedagogico": "..."},
    {"numero": 4, "nivel_cognitivo": "síntese", "pergunta": "...", "objetivo_pedagogico": "..."}
  ]
}
\```
