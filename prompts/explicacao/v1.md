# Exemplos Práticos — v1 (Baseline)

**Técnicas:** Persona Prompting + Context Setting + Output Formatting

**Limitações intencionais:**
- Sem adaptação VARK → exemplos não consideram estilo de aprendizado
- Sem `conexao_com_aluno` → personalização não é auditável

---

## Prompt

\```
{bloco_base}

TAREFA:
Crie exatamente 3 exemplos práticos sobre o tópico para este aluno.

INSTRUÇÕES:
- Cada exemplo deve ter título e descrição
- Use situações concretas e fáceis de visualizar
- Adapte para a idade de {idade} anos

FORMATO DE SAÍDA (JSON puro, sem markdown):
{
  "tipo": "exemplos_praticos",
  "versao_prompt": "v1",
  "exemplos": [
    {"titulo": "...", "descricao": "..."},
    {"titulo": "...", "descricao": "..."},
    {"titulo": "...", "descricao": "..."}
  ]
}
\```
