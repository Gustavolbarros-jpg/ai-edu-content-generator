# Resumo Visual — v2 (Otimizado)

**Técnicas:** Persona + Context Setting + **Structured Prompting** + Adaptação VARK

**Ganho vs v1:** hierarquia de 3 níveis obrigatória torna o mapa navegável. `dica_de_uso` personalizada transforma o mapa em ferramenta de estudo ativa.

---

## Prompt

\```
{bloco_base}

TAREFA:
Crie um resumo visual completo do tópico em formato de mapa mental ASCII.

ESTRUTURA OBRIGATÓRIA DO MAPA:
  Nível 0 (centro): o tópico principal
  Nível 1 (ramos): os 3 ou 4 conceitos principais
  Nível 2 (sub-ramos): 2 detalhes ou exemplos para cada conceito do Nível 1
  Nível 3 (folhas, opcional): conexões ou curiosidades

ADAPTE ao estilo "{estilo_aprendizado}":
  visual        → use símbolos e estrutura clara com espaçamento generoso
  auditivo      → adicione palavras-chave sonoras ou rítmicas nos nós
  leitura-escrita → inclua definições curtas junto aos nós
  cinestésico   → inclua verbos de ação nos nós ("fazer", "testar", "observar")

REGRAS: use │ ├ └ ─ ┬ — cada nó em uma linha — indentação indica hierarquia

FORMATO DE SAÍDA OBRIGATÓRIO (JSON puro, sem markdown):
{
  "tipo": "resumo_visual",
  "versao_prompt": "v2",
  "estilo_aplicado": "{estilo_aprendizado}",
  "mapa_ascii": "mapa completo aqui",
  "legenda": "explique as relações principais do mapa",
  "dica_de_uso": "como o aluno pode usar este mapa para estudar"
}
\```
