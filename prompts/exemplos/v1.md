    # Explicação Conceitual — v1 (Baseline)

**Técnicas:** Persona Prompting + Context Setting + Output Formatting

**Limitações intencionais:**
- Sem chain-of-thought → modelo escolhe analogias sem critério pedagógico
- Sem campos semânticos extras → menos auditável que v2

---

## Prompt

\```
{bloco_base}

TAREFA:
Escreva uma explicação conceitual sobre o tópico acima para este aluno.

INSTRUÇÕES:
- Adapte a linguagem para {idade} anos e nível {nivel_conhecimento}
- Use pelo menos uma analogia para facilitar a compreensão
- Cubra os conceitos principais do tópico

FORMATO DE SAÍDA (JSON puro, sem markdown):
{
  "tipo": "explicacao_conceitual",
  "versao_prompt": "v1",
  "titulo": "título atrativo para o aluno",
  "conteudo": "explicação completa aqui"
}
\```
