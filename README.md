# ai-edu-content-generator
##  Objetivo
Este projeto tem como objetivo criar uma plataforma educacional baseada em prompts,
voltada para estudo, explicação conceitual e prática guiada em programação e tecnologia.

O foco inicial é o aprendizado estruturado, com prompts organizados por nível,
tema e objetivo pedagógico.

---

## O que este projeto é
- Uma plataforma educacional orientada a prompts
- Um exercício de arquitetura, organização e versionamento
- Um ambiente para treinar boas práticas de desenvolvimento

---

##  O que este projeto NÃO é (por enquanto)
- Não é uma aplicação completa
- Não possui lógica de negócio implementada
- Não possui integração com banco de dados
- Não possui API nem interface final

---

##  Estrutura inicial do projeto
- `src/` → Código-fonte (a ser implementado)
- `prompts/` → Conteúdo educacional baseado em prompts
- `templates/` → Templates reutilizáveis
- `data/` → Dados estáticos ou exemplos
- `caches/` → Cache temporário
